Некоторые коментарии к проекту.

ez.css и ez.js - это стили и функции JS которые применимы ко всему сайту. Т.е. на каждой странице будут применяться эти стили и есть место для этих функций.

Про БД
В БД тесно переплетены таблицы бизнес логики и самого сайта. Так есть таблицы пользователей (users) и сесий (sessions). При каждой авторизации в таблицу сессий создаеться УИД, он же и возвращаеться куками на клиента, в дальнейшем по этому УИДУ происходит идентификация всех операций.
